Detailed Description

Contains constant values used in the Tucker decomposition algorithm.

This namespace defines a set of constant values that are used throughout the Tucker decomposition implementation. These constants are used to configure various aspects of the algorithm, such as the maximum order of tensors, the maximum number of iterations, and regularization parameters.
