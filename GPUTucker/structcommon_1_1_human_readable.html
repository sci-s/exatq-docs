Detailed Description

Converts a size in bytes to a human-readable format.

The HumanReadable struct provides a way to convert a size in bytes to a more readable format, such as KB, MB, GB, etc., depending on the size. It supports outputting the size in a concise format using the << operator.

This struct is useful when you want to display sizes in a format that is easier to read and understand. For instance, instead of displaying a size as "1048576 bytes," it can be displayed as "1MB (1048576 bytes)".
