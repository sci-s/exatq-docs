Detailed Description

CUDA agent class for managing CUDA resources in Tucker decomposition.

The CudaAgent class is responsible for managing CUDA devices, streams, and memory regions in the Tucker decomposition program. It handles the allocation of device buffers, creation of CUDA streams, and manages interactions with the GPU during tensor operations.

Template Parameters
TensorTypeThe type of the tensor to be managed by the CUDA agent.

The class provides methods to allocate device buffers, set up CUDA streams, and manage memory regions required for efficient GPU computations. It also provides access to device properties and manages multiple CUDA streams for concurrent operations.
