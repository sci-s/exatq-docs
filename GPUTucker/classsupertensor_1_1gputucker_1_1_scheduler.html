Detailed Description

Scheduler class for Tucker decomposition.

This class is responsible for scheduling the tensor data and optimizer tasks across multiple GPU devices during Tucker decomposition. It partitions the tensor data and assigns tasks to different GPUs based on the chosen partitioning strategy.

Template Parameters
TensorTypeThe type of tensor being decomposed.
OptimizerTypeThe type of optimizer used for partitioning and task scheduling.

The Scheduler class manages the distribution of computation tasks across GPUs, either by partitioning based on non-zero elements or by tensor dimensions. It ensures that each GPU is effectively utilized for the decomposition process.
