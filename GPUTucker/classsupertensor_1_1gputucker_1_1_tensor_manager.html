Detailed Description

Tensor manager class for Tucker decomposition.

This class is responsible for managing the tensor data used in the Tucker decomposition program. It handles reading tensor data from files and creating tensor blocks that are necessary for the decomposition process.

Template Parameters
TensorTypeThe type of tensor being managed.

The TensorManager class provides methods to read tensor data from a file and to create tensor blocks based on the data and an optimizer. These blocks are then used in the Tucker decomposition to efficiently process the tensor.
