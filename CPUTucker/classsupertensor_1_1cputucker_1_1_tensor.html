void supertensor::cputucker::Tensor::AssignIndicesOfEachBlock ( )

Assigns the blocks' metadata(where_nnz, count_nnz) for each mode(axis) in the tensor.

This function calculates and assigns the locations and count of non-zero elements within each block, across all modes (axes) of the tensor.

Exceptions
AssertionIf the block count is zero.

◆ MakeBlocks()

void supertensor::cputucker::Tensor::MakeBlocks ( uint64_t new_block_count,
uint64_t * histogram )

Creates blocks for the tensor based on the given histogram and block count.

Allocates and initializes blocks according to the specified block count and non-zero element distribution provided in the histogram.

Parameters
new_block_countThe number of blocks to create.
histogramArray containing the number of non-zero elements in each block.
Exceptions
AssertionIf the block count does not match the specified count.
