#define SCHEDULER_TEMPLATE   template <typename TensorType>

Scheduler class for managing task distribution in Tucker decomposition.

The Scheduler class is responsible for organizing and distributing tasks (tensor blocks) based on the number of non-zero elements in each block. This allows for efficient processing of tensor blocks during the Tucker decomposition.
