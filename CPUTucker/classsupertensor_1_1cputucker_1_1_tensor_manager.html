Detailed Description

The TensorManager class manages tensor data and performs I/O operations.

Manages tensor data and intermediate data generated during Tucker decomposition and performs I/O operations for them. The TensorManager class is responsible for reading and writing tensor data from/to storages. The class ensures that the tensor data is correctly correctly parsed, stored and written to storages during Tucker decomposition process.
